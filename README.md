# Exploratory Data Analysis (EDA) on Books Dataset

## Project Overview

This project focuses on Exploratory Data Analysis (EDA) of a books dataset collected through web scraping.

The objective is to understand the structure of the dataset, identify patterns, analyze pricing trends, and generate meaningful insights using statistical methods and visualizations.

## Technologies Used

* Python
* Pandas
* Matplotlib
* Jupyter Notebook

## Dataset Information

The dataset contains information about books including:

* Title
* Price
* Availability Status

Total Records: Approximately 1000 Books

## Analysis Performed

### 1. Dataset Inspection

* Checked dataset dimensions
* Examined data types
* Reviewed dataset structure

### 2. Data Quality Assessment

* Checked for missing values
* Verified data consistency

### 3. Statistical Analysis

* Calculated average book price
* Identified highest priced books
* Identified lowest priced books
* Generated summary statistics

### 4. Price Analysis

* Analyzed price distribution
* Compared expensive and affordable books
* Examined overall pricing trends

### 5. Data Visualization

* Histogram of book prices
* Box plot analysis
* Price distribution insights

## Key Findings

* The dataset contains approximately 1000 books.
* No major missing values were identified.
* Book prices vary across a wide range.
* Most books are concentrated in the mid-price range.
* The dataset is suitable for further visualization and predictive analysis.
